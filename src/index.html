RiffReady

As part of my final year of university, I created a web application that enables guitarists to play along to songs by linking Spotify and Ultimate Guitar. I used Spotify's Web Developer SDK to link to a user's account and retrieve playback information, and then scraped Ultimate Guitar to retrieve the most appropriate tablature or chord sheet using that data. I am currently working on making this application publically available.

Letterboxd Alert

I created a python script to take a user's letterboxd watchlist, downloaded as a csv from Letterboxd's website, to set up an alert for when each film becomes available on a user's preferred streaming service. The program uses the Watchmode API to extract information about each film, which is then stored in a local csv file for the program to monitor for the alert system.
